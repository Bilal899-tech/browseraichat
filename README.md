# BrowserAI Chat — nexagaze project

> Built by Founder Bilal

Browser-based AI chatbot using Transformers.js (flan-t5-small). Runs 100% client-side, no server needed.

## Tech Stack
- Transformers.js (Xenova/flan-t5-small)
- HTML / CSS / JavaScript
- Works offline after model download

## Setup
```bash
npm install
npm start
```

## Features
- 100% browser-based AI chatbot
- No backend or server required
- Uses flan-t5-small for text generation
- Progress indicator for model download
- Clean, modern dark UI

## 📖 Documentation

### Architecture
Single HTML file. All logic runs in the browser via Transformers.js CDN import map. 
No backend server required. Model loads from Hugging Face Hub on first use (cached locally).

### Model
- **Pipeline:** text2text-generation
- **Model:** Xenova/flan-t5-small
- **Weights:** ~80MB (downloaded once, cached in browser)

### Usage
1. Open the page
2. Wait for model download (progress bar shown)
3. Type a question in the chat input
4. AI generates response in real-time
